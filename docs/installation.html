NPM Installation
Recommended for modern web applications
# Install via NPM npm install @niel-blanca/signature-pad # Or using Yarn yarn add @niel-blanca/signature-pad # Or using PNPM pnpm add @niel-blanca/signature-pad

‚úÖ Advantages

  • Version control and dependency management
  • Tree shaking support (reduce bundle size)
  • TypeScript definitions included
  • Works with bundlers (Webpack, Vite, etc.)
  • Offline development support

‚ùå Considerations

  • Requires build process
  • Need Node.js and package manager
  • Slightly more complex setup

üìã Step-by-Step Setup

1

Install the Package

Run the installation command in your project directory

npm install @niel-blanca/signature-pad
2

Import the Library

Import SignaturePad in your JavaScript/TypeScript file

// ES6 Module Import (recommended) import SignaturePad from '@niel-blanca/signature-pad'; // CommonJS (Node.js style) const SignaturePad = require('@niel-blanca/signature-pad');
3

Create HTML Container

Add a container element for the signature pad

<div id="signature-container" style="width: 100%; height: 300px; border: 1px solid #ccc;"></div>
4

Initialize the Signature Pad

Create a new instance and start using it

const container = document.getElementById('signature-container'); const signaturePad = new SignaturePad(container); // Test if it's working console.log('Signature pad created successfully!');

CDN Installation
Quick setup for prototyping and simple projects
<!-- Include from UNPKG CDN --> <script src="https://unpkg.com/@niel-blanca/signature-pad@latest/dist/signature-pad.min.js"></script> <!-- Or from JSDelivr CDN --> <script src="https://cdn.jsdelivr.net/npm/@niel-blanca/signature-pad@latest/dist/signature-pad.min.js"></script>

‚úÖ Advantages

  • No build process required
  • Works directly in browser
  • Perfect for prototyping
  • Global CDN for fast loading
  • Easy to get started

‚ùå Considerations

  • Requires internet connection
  • No version locking by default
  • Global namespace usage
  • Less control over caching

üìã Step-by-Step Setup

1

Include the Script Tag

Add the CDN script tag to your HTML head or before closing body tag

<script src="https://unpkg.com/@niel-blanca/signature-pad@latest/dist/signature-pad.min.js"></script>
2

Create HTML Container

Add a container element for the signature pad

<div id="my-signature-pad" style="width: 100%; height: 300px; border: 1px solid #ccc;"></div>
3

Initialize After DOM Load

Use the global SignaturePad class to create an instance

// Wait for DOM to be ready document.addEventListener('DOMContentLoaded', function() { const container = document.getElementById('my-signature-pad'); const signaturePad = new SignaturePad(container); console.log('CDN signature pad ready!'); });
