Basic Usage Guide

Learn the fundamentals of SignaturePad.js with step-by-step examples and interactive demos. Each example includes working code and a live signature pad you can test immediately.

1. Getting Started - Your First Signature Pad

The most basic implementation requires just a canvas element and a few lines of JavaScript:

HTML Structure
<canvas id="signature-pad" width="400" height="200"></canvas>
<button id="clear">Clear</button>
JavaScript Implementation
// Initialize the signature pad
const canvas = document.getElementById('signature-pad');
const signaturePad = new SignaturePad(canvas);

// Add clear functionality
document.getElementById('clear').addEventListener('click', () => {
    signaturePad.clear();
});

2. Configuration Options

Customize your signature pad behavior with configuration options passed to the constructor:

Configuration Example
const signaturePad = new SignaturePad(canvas, {
    backgroundColor: 'rgb(255, 255, 255)', // Background color
    penColor: 'rgb(0, 0, 0)',             // Pen color
    minWidth: 0.5,                        // Minimum stroke width
    maxWidth: 2.5,                        // Maximum stroke width
    throttle: 16,                         // Throttle drawing in ms
    minDistance: 5,                       // Minimum distance between points
    dotSize: 1.0,                         // Size of dots for very short strokes
    velocityFilterWeight: 0.7             // Velocity filtering weight
});

3. Essential Methods

These are the most commonly used methods you'll need for basic signature pad functionality:

  • clear() - Clears the signature pad canvas
  • isEmpty() - Returns true if the canvas is empty
  • toDataURL() - Exports signature as base64 PNG image
  • fromDataURL() - Loads a signature from base64 data
  • on() / off() - Event listener management
    • Common Method Examples
    // Check if signature exists before saving
if (!signaturePad.isEmpty()) {
    const dataURL = signaturePad.toDataURL(); // Get as PNG
    console.log('Signature data:', dataURL);
} else {
    alert('Please provide a signature first.');
}

// Load a previously saved signature
signaturePad.fromDataURL('data:image/png;base64,iVBORw0...');

// Listen to drawing events
signaturePad.on('beginStroke', () => console.log('Started drawing'));
signaturePad.on('endStroke', () => console.log('Finished stroke'));

    4. Responsive Design

    Make your signature pad work seamlessly across all devices with proper responsive implementation:

    CSS for Responsive Canvas
    .signature-canvas {
    border: 2px solid #ddd;
    border-radius: 8px;
    width: 100%;
    max-width: 500px;
    height: auto;
    touch-action: none; /* Prevents scrolling on touch devices */
}

@media (max-width: 768px) {
    .signature-canvas {
        max-width: 100%;
        height: 200px; /* Fixed height on mobile */
    }
}
    JavaScript for Auto-sizing
    function resizeCanvas(canvas) {
    const ratio = Math.max(window.devicePixelRatio || 1, 1);
    const rect = canvas.getBoundingClientRect();
    
    canvas.width = rect.width * ratio;
    canvas.height = rect.height * ratio;
    
    const ctx = canvas.getContext('2d');
    ctx.scale(ratio, ratio);
    
    // Reinitialize SignaturePad after resize
    signaturePad.clear();
}

// Resize on window resize
window.addEventListener('resize', () => resizeCanvas(canvas));

    5. Form Integration

    Integrate signature pads seamlessly with HTML forms for data collection:

    HTML Form Structure
    <form id="signature-form">
    <label for="name">Full Name:</label>
    <input type="text" id="name" name="name" required>
    
    <label>Signature:</label>
    <canvas id="signature-canvas" width="400" height="200"></canvas>
    <input type="hidden" id="signature-data" name="signature">
    
    <button type="button" onclick="clearSignature()">Clear</button>
    <button type="submit">Submit Form</button>
</form>
    Form Handling JavaScript
    const form = document.getElementById('signature-form');
const signatureInput = document.getElementById('signature-data');

form.addEventListener('submit', (e) => {
    e.preventDefault();
    
    if (signaturePad.isEmpty()) {
        alert('Please provide a signature before submitting.');
        return;
    }
    
    // Store signature data in hidden input
    signatureInput.value = signaturePad.toDataURL();
    
    // Now submit the form (or handle data as needed)
    console.log('Form data ready:', new FormData(form));
});

function clearSignature() {
    signaturePad.clear();
    signatureInput.value = '';
}
