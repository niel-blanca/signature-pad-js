1. Getting Started - Your First Signature Pad

The most basic implementation requires just a canvas element and a few lines of JavaScript.

<canvas id="signature-pad" width="400" height="200"></canvas> <button id="clear">Clear</button>
// Initialize the signature pad const canvas = document.getElementById('signature-pad'); const signaturePad = new SignaturePad(canvas); // Add clear functionality document.getElementById('clear').addEventListener('click', () => { signaturePad.clear(); });

3. Line Thickness Control

Control the thickness of signature strokes for different writing styles.

const signaturePad = new SignaturePad(canvas, { minWidth: 0.5, // Minimum line width maxWidth: 2.5, // Maximum line width velocityFilterWeight: 0.7 // Smoothness factor }); // Adjust thickness dynamically signaturePad.minWidth = 1; signaturePad.maxWidth = 4;

4. Essential Methods

Key methods for controlling and interacting with the signature pad.

// Check if pad is empty if (signaturePad.isEmpty()) { console.log('Please provide a signature'); } // Clear the signature signaturePad.clear(); // Get signature as data URL const dataURL = signaturePad.toDataURL(); // PNG by default const jpegURL = signaturePad.toDataURL('image/jpeg'); // Enable/disable drawing signaturePad.off(); // Disable signaturePad.on(); // Enable

5. Responsive Design

Make signature pads work perfectly on all device sizes.

function resizeCanvas() { const canvas = document.getElementById('signature-pad'); const ratio = Math.max(window.devicePixelRatio || 1, 1); canvas.width = canvas.offsetWidth * ratio; canvas.height = canvas.offsetHeight * ratio; canvas.getContext('2d').scale(ratio, ratio); signaturePad.clear(); // Clear to avoid scaling issues } window.addEventListener('resize', resizeCanvas); resizeCanvas(); // Initialize

document.getElementById('signature-form').addEventListener('submit', (e) => { if (signaturePad.isEmpty()) { e.preventDefault(); alert('Please provide a signature first.'); return; } // Save signature data to hidden input const dataURL = signaturePad.toDataURL(); document.getElementById('signature-data').value = dataURL; });
