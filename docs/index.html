üöÄ Get Started in Seconds

Add this signature pad to your project with just a few lines of code

Install via NPM

npm install @niel-blanca/signature-pad
// ES6 Import import SignaturePad from '@niel-blanca/signature-pad'; // CommonJS const SignaturePad = require('@niel-blanca/signature-pad');

Use via CDN

<script src="https://unpkg.com/@niel-blanca/signature-pad@latest/dist/signature-pad.min.js"></script>
// The SignaturePad class is now available globally const signaturePad = new SignaturePad(container);

Basic Usage

// Create container element const container = document.getElementById('signature-container'); // Initialize signature pad const signaturePad = new SignaturePad(container, { color: '#000000', thickness: 2, background: '#ffffff' }); // Export as PNG const pngData = signaturePad.toDataURL(); // Export as JSON const jsonData = signaturePad.toJSON(); // Clear the pad signaturePad.clear();
